## 🏢 關於我們

環友科技股份有限公司致力於提供專業的軟體開發與技術服務，專精於企業級應用系統開發與維護。我們擁有豐富的軟體開發經驗，為客戶提供量身訂製的解決方案。

### 🎯 核心服務
- 📱 企業級 Web 應用程式開發
- 🔧 API 服務設計與開發
- 🗄️ 資料庫設計與優化
- ☁️ 雲端服務部署與維護
- 🔄 系統整合與自動化
